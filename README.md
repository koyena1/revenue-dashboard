# Revenue Dashboard

This is a simple internal dashboard created using Next.js 15, Tailwind, and Recharts.

## What It Does

- View company revenue data from 2021 to 2024
- Filter by minimum revenue
- Switch between bar, line, and pie charts

## Technologies

- Next.js (App Router)
- TypeScript
- TailwindCSS
- Recharts

## How to Run

```bash
npm install
npm run dev
